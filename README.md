# Vue.js starter

- Vue.js
- Vue Router
- Jest
- Node Sass
- TailwindCSS
- Storybook
- PurgeCSS
- Prettier

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```

### Run your unit tests

```
yarn run test:unit
```
